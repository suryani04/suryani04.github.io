About Me


Hallo Semua, perkenalkan nama saya Suryani Berutu, lahir pada tanggal 27 April 2001 dan sekarang umur saya hampir menginjak 21 tahun pada tahun ini. Teman-teman dekat saya dan keluarga saya biasa memanggil saya dengan sebutan Ani, saya anak ke-3 dari 5 bersaudara. Ayah saya bekerja sebagai karyawan tetap di salah satu PT pengolahan kelapa sawit yang terdapat didekat tempat tinggal kami dan ibu saya sebagi IRT (ibu rumah tangga). Saya seorang mahasiswa jurusan Teknik Informatika, di Universitas Malikussaleh, tepatnya di Lhoksemawe, Aceh. Saya angkatan 2019 itu arti seharusnya tahun 2023 saya sudah menyelesaikan kuliah saya, semoga saja ya. Sekarang saya baru memasuki semester ke 6 dan sejauh ini alhamdulillah belum ada mata kuliah yang mengulang dan semoga saja tidak akan pernah ada. Dikampus saya yang berkaitan dengan materi full stack web developer haya matakuliah pemograman web dan itupun masih sangat dasar. Tujuan saya mengikuti kampus merdeka ini adalah untuk memperdalam ilmu tentang web developer, terutama yang berkaitan dengan : HTML, PHP, CSS, Laravel dan API. Semoga dengan adanya program kampus merdeka ini bisa membuat saya lebih mengerti dan bisa mendalami tentang web developer
